# Eamon Express Invoice Cost Prediction

Ridge regression model built in R to forecast invoice costs based on weight, service type, and shipment features.  
Includes data cleaning, multicollinearity handling, cross-validation, and residual diagnostics.

**Files**
- `EamonExpress_Report.pdf` – Full report
- `All_Plots_Report.pdf` – Correlation, ridge CV, actual vs predicted, residuals
- `ridge_predictions.csv` – Predicted invoice totals
- `eamonexpress_analysis.R` – Main analysis script
